Amazon Web Services Amazon Route 53 (Route 53) is part of Amazon.coms cloud computing platform, Amazon Web Services (AWS). Route 53 provides scalable and highly available Domain Name System (DNS). The name (Route 53) is a reference to TCP or UDP port 53, where DNS server requests are addressed.

ARIN Reg-RWS ARIN offers public access to ARIN registration data via a number of services. Traditionally, these services are known in the industry as Whois in reference to the public data service of the ARPANET, precursor of todays modern Internet. Whois services are offered by all the Regional Internet Registries (RIRs), most Internet Routing Registries (IRRs) and most domain name registries and registrars.

StatDNS This is a REST API allowing to perform DNS queries over HTTP, and to get reverse PTR records for both IPv4 and IPv6 addresses. It outputs JSON-encoded DNS responses and can be useful in server-side environments where you cannot perform DNS queries locally, for client-side Web Applications (including Web browser extensions), and for DNS operators. There is no rate limit of any sort at the moment, and the service is free for everyone to use.
