Terms of Service (TOS)

A Terms of Service (TOS) for an API can be one of the single most important building blocks for any API ecosystem. The TOS sets the tone for the entire conversation between API provider and consumer.

While API TOS are a critical element, they are only discussed when things go wrong, and there are very few open discussions, tools or services that help API provers and consumers understand and operate in the legal world that are TOS.

As the pace quickens within the API economy, the need for standardization in area of TOS will increase. This project is meant to understand various efforts and bring them together in a single area.
